==========================================

Archive: fvc-dapp-cpanel.zip
Built: 19 November 2025
Network: Base Sepolia (84532), BNB Testnet (97), Polygon Amoy (80002)

DEPLOYMENT STEPS:
-----------------

1. LOGIN TO CPANEL
   - Access your cPanel hosting account
   - Navigate to File Manager

2. EXTRACT FILES
   - Upload fvc-dapp-cpanel.zip to public_html/
   - Right-click → Extract
   - Move contents of 'out' folder to root of public_html/
   - Delete the 'out' folder and zip file

3. VERIFY STRUCTURE
   public_html/
   ├── .htaccess          (mod_rewrite configuration)
   ├── index.html         (home page)
   ├── 404.html
   ├── logo.png
   ├── _next/             (Next.js assets)
   ├── assets/            (static assets)
   └── roadmap/

4. CONFIGURE DOMAIN
   - Ensure domain points to public_html/
   - .htaccess handles routing automatically

5. SSL CERTIFICATE
   - Enable AutoSSL in cPanel
   - Or install Let's Encrypt certificate
