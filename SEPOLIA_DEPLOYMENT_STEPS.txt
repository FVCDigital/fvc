=============================================
STEP 1: CREATE contracts/.env
=============================================

Create the file contracts/.env with these three values:

    DEPLOYER_PRIVATE_KEY=<private_key_of_wallet_with_sepolia_eth>
    ETHEREUM_SEPOLIA_RPC=https://rpc.sepolia.org
    TREASURY_ADDRESS=<your_gnosis_safe_address_on_sepolia>

IMPORTANT NOTES:
- The deployer must be a regular wallet (EOA), NOT the Gnosis Safe.
  Hardhat cannot sign transactions with a Gnosis Safe.
- The deployment script sets the Gnosis Safe as the beneficiary/owner
  of the Sale contract. The Safe receives all funds and controls the contract.
- The deployer wallet needs ~0.05 Sepolia ETH for gas.
- Get Sepolia ETH: https://sepoliafaucet.com or https://www.alchemy.com/faucets/ethereum-sepolia
- The TREASURY_ADDRESS is the Gnosis Safe on Sepolia. This becomes the owner
  of the Sale contract and receives all payments (ETH, USDC, USDT).

=============================================
STEP 2: DEPLOY TO SEPOLIA
=============================================

    cd contracts
    npx hardhat run scripts/deploy-sepolia.ts --network sepolia

The script automatically:
- Deploys Mock USDC and USDT (with 1M minted to deployer for testing)
- Deploys FVC token (ERC-20, 18 decimals)
- Deploys Vesting contract
- Deploys Sale contract with:
    - rate = 50000 ($0.05 per FVC in 6-decimal stablecoin units)
    - ethUsdRate = 2500000000 ($2,500 per ETH in 6-decimal units)
    - cap = $10,000,000
- Grants MINTER_ROLE to Sale and Vesting contracts
- Accepts USDC and USDT as payment tokens
- Configures vesting (purchases >= $50k get 180-day cliff, 730-day vesting)
- Transfers Vesting ownership to Sale contract
- Activates the sale (setActive(true))
- Saves all addresses to contracts/deployments-sepolia.json
- Prints the exact .env values needed for cPanel

After deployment, the Sale contract is fully operational. No Gnosis Safe
transactions are needed to start — the script handles all configuration.

STEP 4: TEST THE PURCHASE FLOW
=============================================

1. Go to fvcdigital.com/buy-fvc
2. Connect MetaMask (must be on Sepolia, chain ID 11155111)
3. The wizard will prompt to switch network if needed

To test USDC/USDT purchases:
- You need Mock USDC/USDT in your test wallet
- The deployer wallet has 1M of each from deployment
- Transfer some to your test wallet, OR
- Call mockUsdc.mint(yourAddress, amount) from the deployer
- The mock contracts are standard ERC-20 with a public mint() function

To test ETH purchases:
- Just use Sepolia ETH (from faucet)
- Select ETH payment method in the wizard
- Enter amount and confirm
- No token approval needed for ETH (simpler flow than USDC/USDT)

After purchase:
- Click "Add FVC to Wallet" button on the confirmation screen
- This calls wallet_watchAsset (EIP-747) to add FVC to MetaMask
- The FVC logo (FVC.svg) is served from fvcdigital.com/assets/images/FVC.svg
- FVC will NOT appear in MetaMask automatically — the user must click the button

Price verification:
- At rate = 50000, $100 USDC buys 2,000 FVC
- At rate = 50000 and ethUsdRate = 2500000000, 1 ETH buys 50,000 FVC
- The wizard shows a live estimate when entering ETH amounts

=============================================
STEP 6: GNOSIS SAFE OPERATIONS (FUTURE)
=============================================

The Sale contract owner is the Gnosis Safe (TREASURY_ADDRESS).
Only the Safe can call these functions after deployment.

To change sale parameters:
1. Go to https://app.safe.global
2. Select your Safe on Sepolia (chain ID 11155111)
3. New Transaction > Transaction Builder
4. Enter the Sale contract address (from deployments-sepolia.json)
5. Use these function signatures:

   Change FVC price:
     Function: setRate(uint256)
     Value for $0.05: 50000
     Value for $0.10: 100000
     Value for $0.25: 250000

   Change ETH/USD rate (update periodically to track market):
     Function: setEthUsdRate(uint256)
     Value for $2,000: 2000000000
     Value for $2,500: 2500000000
     Value for $3,000: 3000000000
     Set to 0 to disable ETH purchases entirely.

   Pause sale:
     Function: setActive(bool)
     Value: false

   Resume sale:
     Function: setActive(bool)
     Value: true

   Update cap:
     Function: setCap(uint256)
     Value for $10M: 10000000000000
     Value for $20M: 20000000000000

   Accept a new token:
     Function: setAcceptedToken(address,bool,uint8)
     Args: <token_address>, true, <decimals>

6. Submit and confirm with required signers
