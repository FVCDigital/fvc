# Integrations

This directory contains standalone components and integration examples that can be used by external websites and applications to integrate with the FVC Protocol.

## Contents

### `private-sale-card-standalone/`
A standalone JavaScript component for the FVC Protocol Private Sale Card that can be easily integrated into any website. This is separate from the main dapp's PrivateSaleCard component and is designed for external integration.

**Features:**
- Pure JavaScript implementation (no dependencies)
- Responsive design
- Customizable theming
- Ready for external website integration

**Files:**
- `PrivateSaleCard.js` - Main component file
- `demo.html` - Live demo
- `integration-example.html` - Integration example
- `README.md` - Detailed integration guide

## Usage

These components are designed to be:
1. **Standalone** - No external dependencies required
2. **Portable** - Easy to copy and integrate into other projects
3. **Customizable** - Configurable to match your website's design
4. **Production Ready** - Tested and ready for live deployment

## Integration

To use any of these components:
1. Copy the component files to your project
2. Include the JavaScript file in your HTML
3. Initialize the component with your desired configuration
4. Customize the styling as needed

For detailed integration instructions, see the README within each component directory.
