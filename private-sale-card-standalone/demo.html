Features

  • Coming Soon State: All buttons are disabled and show "Coming Soon"
  • Progressive Pricing: Shows the 4-tier pricing structure ($0.025 → $0.10)
  • Vesting Schedule: Displays 12-month cliff and 24-month linear vesting
  • Responsive Design: Works on all device sizes
  • Customisable Theme: Easy to match your website's design
  • No Dependencies: Pure JavaScript, no external libraries required

Integration Guide

Follow these simple steps to integrate the Private Sale Card into your website:

Quick Integration Steps

Download the files: Get the PrivateSaleCard.js file from this package
Add the script to your HTML: Include the JavaScript file in your page
Create a container: Add a div element where you want the card to appear
Initialise the component: Create an instance of the card

HTML Setup

<!DOCTYPE html> <html> <head> <title>Your Website</title> <!-- Import Inter font for best typography --> <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet"> </head> <body> <!-- Container for the Private Sale Card --> <div id="fvc-private-sale"></div> <!-- Include the Private Sale Card script --> <script src="PrivateSaleCard.js"></script> <!-- Initialise the component --> <script> // Create the Private Sale Card const privateSaleCard = new FVCPrivateSaleCard('fvc-private-sale'); </script> </body> </html>

Advanced Usage

// Customise the theme colours const privateSaleCard = new FVCPrivateSaleCard('fvc-private-sale', { theme: { modalBackground: '#1E1E1E', // Custom background primaryText: '#FFFFFF', // Custom text colour generalButton: '#FF6B6B', // Custom accent colour // ... other theme options } }); // Update the card later when functionality is ready privateSaleCard.update({ progress: 25, // 25% progress currentPrice: 0.05, // $0.05 per FVC milestone: 'Early Adopters' }); // Remove the card if needed privateSaleCard.destroy();
