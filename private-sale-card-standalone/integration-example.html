FVC Private Sale Card

Minimal Integration Example

What You Need

Just 3 files: PrivateSaleCard.js, a container div, and 2 lines of JavaScript!

1Add Container to Your HTML

Create a div element where you want the Private Sale Card to appear:

<!-- Add this anywhere in your HTML body --> <div id="fvc-private-sale"></div>

2Include the JavaScript File

Add the script tag before the closing body tag:

<!-- Add this before </body> --> <script src="PrivateSaleCard.js"></script>

3Initialise the Component

Add this JavaScript code to create the card:

<script> // Create the Private Sale Card const privateSaleCard = new FVCPrivateSaleCard('fvc-private-sale'); </script>

4Complete Example

Here's the complete minimal HTML file:

<!DOCTYPE html> <html> <head> <title>My Website</title> <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet"> </head> <body> <!-- Your existing content --> <h1>Welcome to My Website</h1> <!-- FVC Private Sale Card will appear here --> <div id="fvc-private-sale"></div> <!-- Include the component --> <script src="PrivateSaleCard.js"></script> <!-- Initialise the component --> <script> const privateSaleCard = new FVCPrivateSaleCard('fvc-private-sale'); </script> </body> </html>

5Customise Theme (Optional)

Match the card colours to your website:

const privateSaleCard = new FVCPrivateSaleCard('fvc-private-sale', { theme: { modalBackground: '#1E1E1E', // Card background generalButton: '#FF6B6B', // Accent colour primaryText: '#FFFFFF', // Text colour // ... customise other colours } });

6That's It!

You now have a fully functional Private Sale Card on your website! The component:

  • Automatically adapts to different screen sizes
  • Shows all FVC Protocol information
  • Displays "Coming Soon" state (no functional buttons)
  • Can be easily updated when the sale goes live
  • Requires no external dependencies
